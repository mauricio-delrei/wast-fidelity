# Wast Fidelity

A modern church management system built with Java, Spring Boot, PostgreSQL, Redis, and Docker.

## Features

- Member management
- Church secretariat workflows
- Event management
- Financial administration
- Authentication and authorization
- REST APIs

## Technology Stack

- Java 21
- Spring Boot
- PostgreSQL
- Redis
- Docker & Docker Compose
- Flyway
- Maven
